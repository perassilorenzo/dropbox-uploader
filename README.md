# FileCopier

Semplice strumento Windows per copiare un file da un percorso a un altro, tramite finestre di dialogo grafiche.

## Come funziona

1. Si apre una finestra per scegliere il file da copiare
2. Si apre una seconda finestra per scegliere la cartella di destinazione
3. Il file viene copiato nella cartella scelta (sovrascrive se esiste già)
4. La finestra rimane aperta fino a quando non premi un tasto
